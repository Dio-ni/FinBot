# 🤖 FinBot — Telegram-бот для питчинга

Демо-бот, имитирующий оплату коммунальных услуг. RU + KZ языки.

---

## 🚀 Деплой на Railway.app (бесплатно)

### Шаг 1 — Получи токен бота
1. Открой Telegram → найди [@BotFather](https://t.me/BotFather)
2. Напиши `/newbot` → придумай имя → получи токен вида `123456:ABC-DEF...`

### Шаг 2 — Задеплой на Railway

1. Зайди на [railway.app](https://railway.app) → Sign in with GitHub
2. Нажми **"New Project"** → **"Deploy from GitHub repo"**
3. Загрузи эти файлы в GitHub репозиторий (или используй Railway CLI)
4. В Railway зайди в **Variables** → добавь:
   ```
   BOT_TOKEN = твой_токен_от_BotFather
   ```
5. Railway автоматически запустит бота через `Procfile`

### Шаг 3 — Проверь
Открой своего бота в Telegram → нажми `/start`
